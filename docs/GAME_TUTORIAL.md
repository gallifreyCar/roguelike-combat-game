# Blood Cards 游戏教程

## 游戏概述

Blood Cards 是一款肉鸽卡牌战斗游戏，灵感来自 Inscryption。玩家通过构建牌组、放置卡牌、战斗敌人，在三层地图中前进，最终击败Boss获得胜利。

## 核心玩法流程

### 1. 牌组构建（Deck Builder）

在开始冒险前，需要选择15张卡牌组成初始牌组：

- **解锁卡牌**：通过击败Boss解锁新卡牌
- **选牌限制**：同ID卡牌最多选2张
- **随机填充**：点击按钮自动选择15张卡牌
- **保存牌组**：保存当前配置以便下次使用

**策略建议**：
- 保持低费卡牌充足（1-2费）
- 平衡攻击和防御
- 考虑体系联动（见下文）

### 2. 地图探索（Map）

地图由多层节点组成：

| 节点类型 | 图标 | 说明 |
|---------|------|------|
| 战斗 | ⚔️ | 普通敌人战斗 |
| 精英 | ★ | 更强敌人，更好奖励 |
| Boss | 👹 | 章节Boss，击败解锁新卡 |
| 商店 | 💰 | 用金币购买卡牌 |
| 融合 | 🔮 | 合成两张卡牌 |
| 奖励 | 🎁 | 免费获得卡牌或金币 |
| 事件 | ❓ | 随机剧情事件 |

**操作**：
- 点击节点前进
- 只能选择相邻的下一层节点
- **ESC** - 返回主菜单
- **D键** - 查看牌库
- **A键** - 查看成就

### 3. 战斗系统（Combat）

战斗是游戏的核心，分为两个阶段：

#### 出牌阶段（Play Phase）
1. 从手牌区**拖拽**卡牌到棋盘空位
2. 支付卡牌费用（消耗Blood）
3. 右键点击棋盘上的卡牌可以**献祭**获得Blood
4. 点击**BATTLE**按钮开始战斗

#### 战斗阶段（Battle Phase）
卡牌自动进行攻击：
- 玩家卡牌从左到右攻击
- 敌方卡牌从左到右攻击
- 根据攻击力扣除HP
- 当一方棋盘清空后，直接攻击HP

**关键数值**：
- **Cost（费用）**：放置卡牌消耗的Blood
- **ATK（攻击力）**：每次攻击造成的伤害
- **HP（生命值）**：卡牌被消灭前能承受的伤害

### 4. 奖励选择（Reward）

战斗胜利后，选择一项奖励：

| 奖励类型 | 说明 |
|---------|------|
| 🃏 卡牌 | 添加一张卡牌到牌组 |
| 💰 金币 | 获得金币用于商店购买 |
| ❤️ 属性 | 永久提升最大HP |

**Boss奖励**：
- 解锁新卡牌
- 大量金币
- 属性提升

## 进阶机制

### 体系联动（Family System）

卡牌分为6个体系，同体系的卡牌在场上会获得加成：

| 体系 | 中文名 | 加成效果 |
|------|--------|---------|
| flying | 飞行 | +1 ATK |
| beast | 野兽 | +2 HP |
| insect | 昆虫 | 先攻效果 |
| reptile | 爬行 | 护盾 |
| ocean | 海洋 | 击杀回血 |
| mythic | 神话 | 综合加成 |

**策略**：集中选择同一体系的卡牌，形成协同效应。

### 印记系统（Sigils）

部分卡牌拥有特殊印记：

| 印记 | 效果 |
|------|------|
| Airborne | 飞行，可以绕过前排攻击 |
| Bifurcated Strike | 双重攻击，攻击两个目标 |
| Bone King | 根据场上死亡单位提升攻击 |
| Burrower | 攻击后移动到空位 |
| Dam Builder | 放置时建造水坝 |
| Fecundity | 放置时生成1/1松鼠 |
| Sharp Quills | 受伤时反弹伤害 |
| Sprinter | 每回合攻击两次 |
| Stoat | 被攻击时减伤 |

### 融合系统（Fusion）

在融合节点，可以将两张卡牌合成为一张更强大的卡牌：

- 新卡牌继承双方属性之和
- 可能获得新的印记
- 保留体系属性

**融合配方**：
- 相同卡牌 + 相同卡牌 = 升级版
- 不同卡牌 = 混合新卡牌

### 局外成长（Meta Progression）

每次通关都会获得永久提升：

| 升级项 | 效果 |
|--------|------|
| HP Boost | 起始HP +2 |
| Gold Boost | 起始金币 +10 |
| Blood Boost | 起始Blood +1 |
| Better Squirrel | 松鼠HP +1 |
| Starting Rare | 开局获得一张稀有卡 |

## 章节系统

游戏分为三个章节：

1. **森林（Forest）** - 第一章
   - 敌人：森林生物（狼、乌鸦等）
   - Boss：森林守护者

2. **海洋（Ocean）** - 第二章
   - 敌人：海洋生物（鱼、螃蟹等）
   - Boss：海洋领主

3. **天空（Sky）** - 第三章
   - 敌人：飞行生物（鹰、龙等）
   - Boss：天空之王

## 常见问题

### Q: 游戏卡死怎么办？
A: 已修复。原因是帧率未限制导致CPU占用过高。

### Q: ESC为什么会退出整个游戏？
A: 已修复。现在ESC在主菜单不做任何事，其他场景正常返回。

### Q: 战斗胜利后选择奖励无效？
A: 已修复。现在奖励会正确应用到牌组和金币。

### Q: 商店购买无效？
A: 检查是否在Shop标签页，点击BUY按钮购买。

### Q: 如何解锁更多卡牌？
A: 击败章节Boss会解锁新卡牌。

## 游戏平衡分析

### 当前问题

1. **牌组构建闭环不完整**
   - 解锁卡牌后，需要在牌组构建场景选择
   - 目前只能从主菜单进入牌组构建

2. **金币用途有限**
   - 只能在商店购买卡牌
   - 建议增加：移除卡牌费用、回复HP等

3. **融合系统复杂度**
   - 融合配方不直观
   - 建议增加融合预览

4. **新手引导不足**
   - 没有教程场景
   - 建议增加交互式教程

### 改进建议

1. **增加牌组构建入口**

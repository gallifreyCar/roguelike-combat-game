# Blood Cards 🃏

**LÖVE (Love2D)** 와 **Lua** 로 구축된 덱 빌딩 로그라이크 오토 배틀러 게임.

**[English](../README.md)** | **[中文](./README_CN.md)** | **[日本語](./README_JA.md)** | **한국어**

---

## 특징

- 🎴 **카드 배치**: 카드를 4칸 보드에 드래그
- ⚔️ **오토 배틀**: 카드가 자동으로 공격
- 💀 **희생 시스템**: 우클릭으로 Blood 자원 획득
- 🎯 **인장 시스템**: 13가지 고유 능력
- 🔮 **융합 시스템**: 주사위 굴림 리스크 융합
- 🗺️ **로그라이크 맵**: 분기 경로와 다양한 노드 타입
- 🏆 **업적 시스템**: 9개 업적 해금
- 💾 **세이브 시스템**: 진행 상황 자동 저장
- 🌐 **다국어 지원**: 한/영/중/일

## 빠른 시작

```bash
# LÖVE 11.5+ 설치
# macOS
brew install love

# 게임 실행
love .
```

## 플레이 방법

1. **Blood 경제**: 시작 시 1 Blood, 매 턴 +1 (최대 6)
2. **카드 배치**: 핸드에서 보드로 드래그
3. **희생**: 우클릭으로 +1 Blood 획득
4. **배틀**: BATTLE 버튼으로 자동 전투
5. **승리**: 적 HP를 0으로 만들기

| 카드 | 비용 | 공격력 | HP | 특징 |
|------|------|--------|-----|------|

| Stoat | 1 | 1 | 2 | 기본 공격 |
| Wolf | 2 | 2 | 2 | 밸런스형 |
| Bullfrog | 1 | 1 | 4 | 탱크, 강인 인장 |
| Adder | 2 | 1 | 2 | 독 공격 |
| Raven | 2 | 2 | 3 | 비행 인장 |
| Grizzly | 3 | 4 | 6 | 강타형 |
| Mantis | 3 | 3 | 2 | 더블 스트라이크 |
| Cat | 2 | 1 | 1 | 불사 (1회 부활) |
| Hydra | 5 | 3 | 4 | 사망 시 분열 |

## 조작

| 키 | 동작 |
|------|------|
| 좌클릭 + 드래그 | 카드 배치 |

| 스페이스 | 배틀 시작 |
| ESC | 뒤로/메뉴 |
| Tab | 모드 전환 |

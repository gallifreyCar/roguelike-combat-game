# Blood Cards 🃏

基于 **LÖVE (Love2D)** 和 **Lua** 构建的卡组构建类肉鸽自动战斗游戏。

**[English](../README.md)** | **中文** | **[日本語](./README_JA.md)** | **[한국어](./README_KO.md)**

---

## 游戏特色

- 🎴 **卡牌放置**：拖拽卡牌到4格棋盘
- ⚔️ **自动战斗**：卡牌自动攻击
- 💀 **献祭系统**：右键点击卡牌获得Blood资源
- 🎯 **印记系统**：13种独特卡牌能力
- 🔮 **融合系统**：骰子随机融合卡牌
- 🗺️ **肉鸽地图**：分支路线，多种节点类型
- 🏆 **成就系统**：9种成就等待解锁
- 💾 **存档系统**：自动保存进度
- 🌐 **多语言**：中/英/日/韩

## 快速开始

```bash
# 安装 LÖVE 11.5+
# macOS
brew install love

# 运行游戏
love .
```

## 游戏玩法

1. **Blood经济**：开局1点Blood，每回合+1（上限6）
2. **放置卡牌**：从手牌拖拽到棋盘格子
3. **献祭**：右键点击场上卡牌获得+1 Blood
4. **战斗**：点击BATTLE按钮自动战斗
5. **胜利**：将敌方HP降为0

## 地图节点

| 节点 | 描述 |
|------|------|
| [!] 战斗 | 与敌人战斗 |
| [E] 精英 | 更强的敌人，更好的奖励 |
| [+] 奖励 | 选择一张新卡牌 |
| [F] 融合 | 合成卡牌 |
| [$] 商店 | 查看牌库 |
| [?] 事件 | 随机遭遇 |
| [BOSS] | Boss战 |

## 融合系统

### 同卡融合
- 融合2张相同卡牌，必定成功
- +1攻击，+2血量，有机会获得新印记

### 骰子融合
- 融合2张不同卡牌，有风险
- 成功率30%-90%不等
- 可创造强力组合卡：
  - **毒狼**：Adder + Wolf
  - **天空猎手**：Raven + Wolf
  - **传说巨兽**：Grizzly + Eagle

## 操作

| 按键 | 动作 |
|------|------|
| 左键拖拽 | 放置卡牌 |
| 右键 | 献祭卡牌 |
| 空格 | 开始战斗 |
| ESC | 返回/菜单 |
| Tab | 切换模式 |

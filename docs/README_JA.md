# Blood Cards 🃏

**LÖVE (Love2D)** と **Lua** で構築されたデッキ構築型ローグライクオートバトラー。

**[English](../README.md)** | **[中文](./README_CN.md)** | **日本語** | **[한국어](./README_KO.md)**

---

## 特徴

- 🎴 **カード配置**：カードを4マスのボードにドラッグ
- ⚔️ **オートバトル**：カードが自動で攻撃
- 💀 **生贄システム**：右クリックでBloodリソース獲得
- 🎯 **刻印システム**：13種類のユニークな能力
- 🔮 **融合システム**：ダイスロールでリスクを伴う融合
- 🗺️ **ローグライクマップ**：分岐ルートと異なるノードタイプ
- 🏆 **実績システム**：9つの実績を解放
- 💾 **セーブシステム**：進行状況の自動保存
- 🌐 **多言語対応**：日/英/中/韓

## クイックスタート

```bash
# LÖVE 11.5+をインストール
# macOS
brew install love

# ゲームを実行
love .
```

## 遊び方

1. **Blood経済**：開始時1 Blood、毎ターン+1（最大6）
2. **カード配置**：手札からボードにドラッグ
3. **生贄**：右クリックで+1 Blood獲得
4. **バトル**：BATTLEボタンで自動戦闘
5. **勝利**：敵のHPを0にする

## 操作

| キー | アクション |
|------|----------|
| 左クリック+ドラッグ | カード配置 |
| 右クリック | 生贄 |
| スペース | バトル開始 |
| ESC | 戻る/メニュー |
| Tab | モード切替 |
